What it does

它陪你做的每一件事

从聊天到记忆，从知识库到音乐，每个模块都被重新设计成温暖、清晰、好上手的样子。

💬

会用工具的对话

聊天时可一键开启「知识库 / 记忆 / 联网」三种能力，流式回答并实时显示它正在调用哪个工具、查到了什么。

📚

你的私人知识库

上传文档、图片、网页，自动解析切片并打标签，回答问题时带出处引用，做到有据可依。

🧠

会成长的记忆

它把你提到的人、事、物抽取成三元组，连成记忆图谱，并定期反思生成关于你的洞察。

🖼️

看得懂的图片

多模态理解上传的图片，自动生成描述与标签，让它们和文字一样可被全局搜索。

🔬

深度研究与定时任务

给一个题目让它多轮检索写报告，或设定 cron 让它每天替你跑简报、做整理。

🎧

懂心情的电台

根据你近期的情绪推荐音乐，内置迷你播放器，现取可播放音源，边聊边听。

Get Started

三步在本地跑起来

前端独立运行；如需完整功能，再按 README 启动后端与四个存储服务。

1

安装前端依赖

进入 web 目录，安装 npm 包。需要 Node.js ≥ 18。

# 进入前端工程
cd web
npm install
2

配置接口地址

复制环境变量模板（默认通过 Vite 代理把 /api 转发到 localhost:8000）。

cp .env.example .env
# VITE_API_BASE=/api
3

启动开发服务器

打开 http://localhost:5173 即可看到全新的 MyFriend。

npm run dev
➜ Local: http://localhost:5173/
